Hi!

I started developing this webpage on June 15th, 2023 with three goals in mind:

  1. 1. To create a fast and unique website.
  2. 2. To use ZERO third party libraries.
  3. 3. To get better at C, JS, and CSS programming.

Design:

To make this website as fast as possible, the backend webserver is written entirely in C. This webserver also uses a hashtable and linked list to store HTTP request data in a LRU cache for maximum speed and efficiency. Additionally, the backend features an API found at the /api endpoint. For transparancy, this API is fully available for any website user to visit and explore.

The frontend is written with nothing more than simple HTML/CSS and vanilla JS. Please note that my initial goal of using zero third party libraries is still upheld. This website was created primarily for me to gain familiarity with a variety of programming concepts, and I felt as though using third party libraries such as jQuery, Bootstrap, or even external C libraries, would only cheat me out of learning the fundementals the right way.

Source:

This website is 100% open source and can be found on github at:

https://github.com/Hecate946/cyrus

An added bonus of not using any external libraries is making the webpage super easy for people to run themselves. As long as you have the gcc compiler installed, running this website locally is as easy as cloning the github and running ./start
